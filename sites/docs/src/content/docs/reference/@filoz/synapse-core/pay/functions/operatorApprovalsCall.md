Create a call to the operatorApprovals function

This function is used to create a call to the operatorApprovals function for use with the multicall or readContract function.
Use [parseOperatorApprovals](/reference/filoz/synapse-core/pay/functions/parseoperatorapprovals/) to transform the contract output into the action output type.

| Parameter | Type | Description |

| `options` | \{ `address`: `` `0x${string}` ``; `chain`: `Chain`; `contractAddress?`: `` `0x${string}` ``; `operator?`: `` `0x${string}` ``; `token?`: `` `0x${string}` ``; \} | [operatorApprovalsCall.OptionsType](/reference/filoz/synapse-core/pay/namespaces/operatorapprovalscall/type-aliases/optionstype/) |
| `options.address` | `` `0x${string}` `` | The address of the account (client) to query approvals for. |
| `options.chain` | `Chain` | The chain to use to make the call. |
| `options.contractAddress?` | `` `0x${string}` `` | Payments contract address. If not provided, the default is the payments contract address for the chain. |
| `options.operator?` | `` `0x${string}` `` | The address of the operator to query. If not provided, the Warm Storage contract address will be used. |
| `options.token?` | `` `0x${string}` `` | The address of the ERC20 token to query. If not provided, the USDFC token address will be used. |
